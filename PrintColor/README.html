Print Color

This is a basic library to print colored text.
Usage:
from PrintColor import Color, printColor, printRainbow printColor(Color.RED, "Here is some red text.") printColor(Color.GREEN, "Here is some green text.") printRainbow("Here is rainbow text.")

Result:

Here is some red text.
Here is some green text.
